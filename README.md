用于Arduino的授时服务
=
### 测试地址：[`http://time.wzgy.ml/`](http://time.wzgy.ml/)<br>
### 返回格式如下<br>
```json
{"year":2016,"month":12,"day":26,"hour":15,"minute":2,"second":24,"weekday":1}
```
#### 其中<br>
1. `year`为`年`<br>
2. `month`为`月`<br>
3. `day`为`日`<br>
4. `hour`为`小时`<br>
5. `minute`为`分钟`<br>
6. `second`为`秒`<br>
7. `weekday`为`星期`<br>
